# Triage Labels

| Label in mattpocock/skills | Label in this tracker | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer must evaluate the ticket |
| `needs-info` | `needs-info` | Waiting for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified for an agent |
| `ready-for-human` | `ready-for-human` | Requires owner or human action |
| `wontfix` | `wontfix` | Will not be actioned |

Record a role as `Triage: <label>` in a local ticket header when triage applies.
